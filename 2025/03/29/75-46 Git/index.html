75-46 Git

类型(Type)规范（核心字段）

类型 使用场景 示例
feat 新增功能/特性 feat(login): 增加微信扫码登录功能
fix 修复Bug fix(payment): 解决信用卡支付金额校验漏洞
docs 文档更新 docs(api): 补全用户注册接口参数说明
style 代码样式调整（不改变逻辑） style(header): 调整导航栏CSS缩进格式
refactor 代码重构（非功能新增/Bug修复） refactor(cache): 将Redis客户端切换为Lettuce实现
test 测试用例相关 test(auth): 添加OAuth2令牌失效的单元测试
chore 构建/依赖/配置变更 chore(deps): 升级Spring Boot至3.2.1版本
ci CI/CD相关变更 ci(gitlab): 添加SonarQube代码质量检测阶段
perf 性能优化 perf(db): 用户查询接口添加Redis二级缓存
build 构建系统/打包工具变更 build(maven): 将打包方式从JAR改为War
revert 回退代码 revert: 撤销误提交的数据库配置（commit 12345a）
