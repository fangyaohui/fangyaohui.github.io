06 局域网

背景是：那天陆哥通过访问我的ip:端口号就可以访问我本地运行的网页信息。因为学过计算机网络，我想这可能跟局域网有关，但是我已经记不太清了，所以我想借此去了解一下公司内部的局域网是如何搭建的，其原理是什么？简单了解一下，不涉及具体的搭建步骤。

广域网&局域网

这就是我们常说的网络，遍及全世界的网络。相对于广域网，其范围更小，即在一定范围内的计算机互联网络；

公网、外网即广域网

私网、内网即局域网

广域网上的每一台机器都有一个或多个IP地址，但是局域网上的每一台电脑都有一个或多个局域网IP地址，不同的局域网的IP地址可以重复，但是同一个局域网的IP地址不能重复。

广域网（WAN、公网、外网）与局域网（LAN、私网、内网）电脑交换数据要通过路由器或网关的NAT（网络地址转换）进行。一般说来，局域网（LAN、私网、内网）内电脑发起的对外连接请求，路由器或网关都不会加以阻拦，但来自广域网对局域网内电脑电脑连接的请求，路由器或网关在绝大多数情况下都会进行拦截。

通常情况下，网关或路由器对内部向外发出的信息不会进行拦截，但对来自外部想进入内部网络的信息则会进行识别、筛选，认为是安全的、有效的，才会转发给内网电脑。正是这种情况的存在，才导致了很多内网用户没有“远程”，速度也不尽如人意。

这是一个简单的例子：

让我们以一个名为ABC科技的公司为例，介绍如何从零开始搭建一个公司内部局域网。

1. 规划网络拓扑：

ABC科技是一家中小型科技公司，拥有30名员工，分布在两个办公室。他们的网络需求包括：

  • 在两个办公室之间共享文件和资源。
  • 提供无线网络覆盖，以便员工能够在办公室内自由移动并连接网络。
  • 连接到互联网，以便员工能够访问外部资源和服务。

2. 获取网络设备：

ABC科技决定购买以下网络设备：

  • 2台企业级路由器：用于连接到互联网并实现两个办公室之间的通信。
  • 2个交换机：用于在每个办公室内连接计算机、打印机等有线设备。
  • 4个无线接入点（AP）：在每个办公室内提供无线网络覆盖。

3. 连接网络设备：

  • 在每个办公室内，安装一个交换机，并将所有有线设备连接到交换机上。
  • 在每个办公室内，安装两个无线接入点，以覆盖整个办公室。

4. 配置网络设备：

  • 配置两台路由器，将它们连接到互联网，并设置防火墙规则以保护公司网络安全。
  • 在交换机上配置 VLAN，将不同部门的设备隔离开来，提高网络安全性和性能。
  • 配置无线接入点，设置无线网络名称（SSID）、安全密钥等，并配置访客网络以供访客使用。

5. 分配 IP 地址：

  • 设计并分配局域网中的 IP 地址范围，使用 DHCP 来自动分配 IP 地址。
  • 确保每个设备都能够正确获取 IP 地址，并能够与其他设备进行通信。

6. 配置网络服务：

  • 部署文件共享服务器，使员工可以在办公室内共享文件和资源。
  • 部署邮件服务器，使员工可以收发电子邮件。
  • 部署Web服务器，用于公司网站和内部应用程序。

7. 测试和维护：

  • 对新建的局域网进行测试，确保各项功能正常运行，并且网络安全性得到有效保护。
  • 定期检查和维护局域网设备，确保其稳定性和性能。
  • 随着公司规模和需求的变化，及时更新和升级网络设备和服务。

通过以上步骤，ABC科技成功搭建了一个公司内部局域网，满足了员工的日常工作需求，并保障了网络安全和性能。
